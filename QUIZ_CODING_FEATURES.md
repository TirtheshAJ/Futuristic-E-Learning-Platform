## 🎮 Quiz Flow

### **Step 1: Quiz Selection**
- Browse all 7 quizzes (one per course)
- See question breakdown: X MCQs + Y Coding
- View difficulty, duration, rewards

### **Step 2: MCQ Section**
1. Answer 5 multiple choice questions
2. Select one option per question
3. Navigate with Previous/Next buttons
4. Automatic transition to Coding section after last MCQ

### **Step 3: Coding Section**
1. **Question displayed on left**:
   - Problem title
   - Description
   - Expected output
   - Hints (💡)
2. **Code editor on right**:
   - Pre-filled template code
   - Write/edit solution
   - Run code to test
   - See output
3. Navigate between coding challenges
4. Submit quiz after last question

### **Step 4: Results**
- **Score calculation**:
  - MCQ: 100% or 0% per question
  - Coding: Partial credit for attempting (50%)
- Pass/Fail based on 70%+ score
- Earn coins and XP if passed
- Option to retake or go back
